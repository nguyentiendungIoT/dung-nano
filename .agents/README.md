# Agent Memory Files

Thu muc nay chua ghi chu dai han cho agent khi quay lai du an trong phien sau.
Chi dung Markdown. Khong dat source code, build output, hoac file cau hinh tool
tu sinh vao day.

## Cach Dung

- Doc `project_context.md` truoc khi tra loi cau hoi ve firmware.
- Doc `working_rules.md` truoc khi sua file.
- Cap nhat `session_notes.md` khi co quyet dinh quan trong hoac nguoi dung doi
  yeu cau.

Nhung file trong thu muc nay la tri nho cho agent, khong phai mot phan cua
firmware chay tren Arduino.
